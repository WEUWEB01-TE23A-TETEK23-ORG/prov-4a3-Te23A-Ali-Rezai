## Inspelning av hela provet krävs för godkänt prov
1. Starta inspelningen i **OBS Studio**
2. Arbeta med uppgiften
3. När du är klar **commit & sync**
4. Stoppa inspelningen i **OBS Studio** och lämna in videon på **Classroom**

## Instruktioner för provet
* Skapa en webbsida för "Skogsgläntan Glamping".
* Sidan skall vara **960px bred och centrerad**.
* Mittendelens **höjd är 405px**
* Skapa en meny med **rollover-effekt** (understruken) och **vald-effekt** (ljusgrön) likt originalet.
* **Identifiera** och välj ett passande typsnitt.
* Skapa en **"tack-ruta"** som visas när man klickar på knappen "Boka ditt tält nu!"

## Krav på kodkvalitet
* Kommentera din kod.
* Indentera din kod så den är tydlig och lätt att läsa.

### CSS-reset
Du kan använda följande CSS-reset för att nollställa alla standardvärden i webbläsaren:
```css
/* Enkel CSS-reset */
html {
    box-sizing: border-box;
}
*, *:before, *:after {
    box-sizing: inherit;
}
body, h1, h2, h3, h4, h5, h6, p, ul {
    margin: 0;
    padding: 0;
}
img {
    max-width: 100%;
    height: auto;
}
```
